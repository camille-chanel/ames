Installation
------------

From Sources:

You can either clone the public repository:

.. code-block:: console

    git clone git://github.com/camille-chanel/ames

Or download the `tarball`_:

.. code-block:: console

    curl  -OL https://github.com/camille-chanel/ames/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    python setup.py install
